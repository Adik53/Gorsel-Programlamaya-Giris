# TODO Projesi

TODO Projesi, kullanıcıların görevlerini yönetmelerine izin veren basit bir uygulamadır. Kullanıcılar yeni görevler oluşturabilir, takip edebilir ve yönetebilirler.

## Özellikler

- Yeni görevler oluşturma.
- Görevleri tamamlandı olarak işaretleme.
- Görevleri düzenleme ve silme.
- Görevleri duruma göre filtreleme (tamamlandı/tamamlanmadı).
- Toplam görev sayısının görüntülenmesi.
- Kullanıcı dostu ve basit bir arayüz.

## Kurulum ve Çalıştırma

1. Depoyu klonlayın:

git clone https://github.com/sizin-kullanıcı-adınız/todo.git


2. Proje dizinine gidin:

cd todo


3. Bağımlılıkları yükleyin:

npm install


4. Uygulamayı başlatın:

npm start


5. Uygulamayı tarayıcıda açın:

http://localhost:3000
